В ходе выполнения работы были рассмотрены две предметные области: автоматизация деятельности фотостудии и автоматизация документооборота при оформлении документов по практике студентов. В обеих частях работы были выявлены проблемы, связанные с ручной обработкой данных, большие временные затраты, вероятность ошибок, сложность хранения и актуализации информации, а также необходимость удобного доступа к данным через web-интерфейс. В рамках работ были выполнены две web-системы, направленные на автоматизацию разных прикладных задач: обработок заявок фотостудии и подготовки докумментов по практике студентов. Обе системы показывают, что использование специализированных web-приложений позволяет повысить удобство работы с данными, снизить количество ошибок и упростить выполнение повторяющихся операций.
