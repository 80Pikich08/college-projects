Была разработана система документооборота на Django, Django REST Framework, React и SQLite. Система позволяет пользователям регистрироваться, авторизовываться, заполнять данные для документов и генерировать DOCX-файлы на основе шаблонов. Реализовано разделение шаблонов для учебной и производственной практики, сохранение последних введённых данных в браузере через localStorage, а также административная панель для управления пользователями, шаблонами и журналом событий. На уровне базы SQLite были добавлены SQL-триггеры, для автоматического журналирования действий, связанных с данными практики, созданием документов и изменением шаблонов.
В рамках обеспечения безопасности в проекте, были использованы стандартные механизмы выбранных технологий. В Django-проекте были реализованы регистрация и авторизация пользователей, разграничение пользовательских и административных функций, обработка данных через API и контроль работы с шаблонами документов. В ходе тестирования были проверены функции обеих систем. Для системы документооборота были проверены регистрация и авторизация пользователей, заполнение формы данных, сохранение информации в localStorage, генерация документов, работа с разными шаблонами, замена шаблонов администратором и срабатывание SQL-триггеров журналирования.
