В проектной части первого сервиса была разработана веб-система фотостудии на Laravel с использованием PostgreSQL и административной панели Filament. Система включает публичную часть сайта, где пользователь может просматривать услуги, тарифы, выполнять поиск, фильтрацию и сортировку, а также отправлять заявку на съёмку. В административной части реализовано управление услугами, тарифами, заявками, клиентами и статистикой. Была спроектирована база данных, добавлена возможность выбора нескольких услуг в одной заявке, реализована загрузка изображений, процедура PostgreSQL для пересчёта статистики клиентов.
В рамках обеспечения безопасности в проекте, были использованы стандартные механизмы выбранных технологий. В Laravel-проекте применялись авторизация для административной панели, CSRF-защита, серверная валидация данных, хранение конфиденциальных параметров в файле .env и ограничение загрузки файлов изображениями. Для проекта фотостудии были протестированы отображения страницы сайта, вывод услуг и тарифов, работа поиска, фильтрации и сортировки, создание заявок, добавление нескольких услуг в одну заявку, загрузка изображений, работа административной панели, выполнение процедур и статистики, а также резервное копирование и восстановление базы данных.
